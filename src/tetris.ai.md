# Tetris SPA v1
This is a Tetris game SPA (Single Page Application) that serves as a proof of
concept for the AI.MD project.

## Design Rules
- Theme: Terminal dark mode (ping background, standard color blocks).
- Render a 10x20 grid Tetris board at the center of the screen. Each block must be exact square shape.

## Functional Requirements
- Users use arrow keys to rotate or move blocks and SPACE key to drop the block.
- The scoreboard must update in real time, increasing by 100 points for each cleared line.
